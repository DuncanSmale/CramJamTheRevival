# Code Styling
## Naming
### Functions:
```javascript
function myFunction(p1, p2){
    return p1*p2;
} 
```
### Variables:
```javascript
let myVariable = 6;
const MY_CONSTANT = 3.14;
```
### Classes:
```javascript
//myClass.js
class MyClass{
    constructor(m1, m2){
        this.m1 = m1;
        this.m2 = m2;
    }
}
```
